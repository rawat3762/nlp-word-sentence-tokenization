# Word & Sentence Tokenization

## 📌 Project Overview
This is a beginner-friendly Natural Language Processing (NLP) project using the NLTK library.  
It demonstrates how unstructured text can be broken into smaller meaningful units like words and sentences using tokenization.

---

## 🚀 Features
- Sentence Tokenization using `sent_tokenize`
- Word Tokenization using `word_tokenize`
- Simple text preprocessing demonstration

---

## 🧠 What is Tokenization?
Tokenization is the process of breaking text into smaller parts called tokens.  
- Sentence Tokenization → Splitting text into sentences  
- Word Tokenization → Splitting sentences into words  

---

## 🛠️ Technologies Used
- Python
- NLTK (Natural Language Toolkit)
